# CoachMate PRD (Product Requirements Document)

---

## 제품 개요

- **제품명**: CoachMate (코치메이트)
- **포지션**: 운동전문가를 위한 AI 코칭 코파일럿
- **핵심 가치**: 기록·피드백을 대신 만들어주는 AI 레이어
- **1순위 타겟**: 헬스 PT 트레이너 (1회 이상 수업 진행한 회원 담당)

---

## 핵심 문제

트레이너의 실력이 기록·소통의 병목 때문에 회원에게 전달되지 못함.
기존 서비스는 "정리해주는 도구"이지, "대신 만들어주는 도구"가 아님.

---

## 설계 원칙

1. **입력 부담 제로** — 타이핑이 아닌 음성/자연어로 기록
2. **회원 앱 설치 불필요** — 카톡/웹링크로 결과 전달
3. **비정형 데이터 수용** — 통증, 컨디션, 메모를 자연어 그대로
4. **일을 줄여주는 도구** — "또 다른 일거리"가 아닐 것
5. **기존 도구 보완** — 센터 CRM, 카톡 등을 대체하지 않음

---

## 1단계 (MVP) 기능

### 기능 1: 운동일지 AI 생성

트레이너가 자연어로 수업 내용을 입력하면, AI가 코치와 회원이 함께 볼 수 있는 운동일지를 생성한다.

**사용자 시나리오:**
1. 수업 직후 트레이너가 홈 화면 바로가기 한 번 터치 → CoachMate 입력 화면
2. 자연어로 수업 내용 입력 (축약어, 비정형 표현 모두 가능)
3. AI가 구조화된 운동일지를 생성
4. 트레이너가 내용 확인 후 필요시 수정
5. 복사 버튼 → 카카오톡에 붙여넣기 → 회원에게 전달

**입력 (트레이너 자연어):**
```
민수님 오늘 스쿼트 80kg 8회 4세트, 벤치프레스 60kg 10회 3세트,
랫풀다운 50kg 12회 3세트 했습니다.
마지막 스쿼트 세트 폼 좀 흐트러졌고, 오른쪽 어깨 약간 불편하다고 해서
다음에 오버헤드 프레스는 빼려고 합니다. 수면 부족으로 컨디션 안 좋았음.
```

**출력 (운동일지 — 코치와 회원이 같이 보는 문서):**
```
📋 운동일지
2026.02.09 | 민수님

🏋️ 오늘의 운동
종목          무게     횟수    세트
스쿼트        80kg    8회     4세트
벤치프레스     60kg    10회    3세트
랫풀다운      50kg    12회    3세트

📝 코치 노트
- 스쿼트 마지막 세트에서 폼이 다소 흐트러짐 → 다음 세션에서 집중 체크
- 오른쪽 어깨 불편감 있어 다음 시간 오버헤드 프레스 제외 예정
- 컨디션: 수면 부족으로 전반적 피로감

다음 세션에서는 어깨 상태 확인 후 상체 프로그램 조정하겠습니다.
수고하셨습니다! 🙌
```

**핵심 설계:**
- 코치와 회원이 같은 문서를 본다 (별도 버전 없음)
- 둘 다 읽을 수 있는 톤 (전문적이면서 이해하기 쉬운)
- 비정형 데이터 (통증, 컨디션, 메모) 자연스럽게 포함
- 유저 테스트 후 톤/포맷 지속 보완

**입력 설계:**
- 회원 이름 (필수)
- 수업 내용 — 자연어 텍스트 박스 (형식 제한 없음)
- 날짜 — 자동으로 오늘 날짜 (수정 가능)
- 축약어 인식: "벤치 60 10 3" → 벤치프레스 60kg 10회 3세트

**공유 설계 (단계별 진화):**

| 단계 | 방식 | 필요 기술 |
|------|------|----------|
| MVP | 복사 버튼 → 카톡 붙여넣기 | 프론트엔드만 |
| 1.5단계 | 카톡 알림톡 + 웹 링크 | 백엔드 + DB + 카카오 비즈메시지 |
| 2단계 | 앱 저장 + 카톡 링크 공유 | 앱 개발 |

- MVP: 회원에게 앱/계정 요구 없음. 트레이너도 로그인 없이 사용 가능
- 1.5단계: 웹 링크로 정돈된 일지 제공. 앱 설치 싫은 회원은 웹으로만 확인
- 2단계: 앱 설치한 회원은 히스토리 열람. 설치 유도하되 강제 아님

**MVP에서 필요한 것:**
- 텍스트 입력 폼 (이름 + 내용 + 날짜)
- AI API 연동 (Claude API)
- 결과 출력 화면 + 복사 버튼
- 모바일 최적화 (트레이너가 폰으로 사용)
- 홈 화면 바로가기 (PWA 또는 웹앱 아이콘)

**MVP에서 필요 없는 것:**
- 회원 계정/로그인
- 데이터베이스 (기록 저장)
- 음성 입력
- 회원 앱
- 카카오 알림톡 연동

## 2단계 기능 (예정)

| 기능 | 설명 |
|------|------|
| 영상 업로드 + 리포트 전송 | 트레이너가 영상 촬영 → 회원에게 리포트와 함께 전달 |
| 영상 데이터 수집 → AI 학습 | 동작 자동 태깅, 자세 분석 (공개 데이터셋 + 자체 데이터) |
| 프로그램 자동 생성 코파일럿 | AI 초안 → 트레이너 수정/승인 |
| 회원 커뮤니케이션 코파일럿 | 회원별 주간 요약·격려 메시지 초안 생성 |
| 이탈 예측 | 데이터 기반 리스크 스코어링 + 추천 액션 |

### 영상 데이터 파이프라인 (2단계)

**기술 요구사항:**
- 클라우드 스토리지 (Cloudflare R2 / AWS S3)
- 포즈 추출 (MediaPipe 등) → 키포인트 데이터만 보관 가능
- 베이스 데이터: AI허브 피트니스 자세 데이터셋, M3GYM, Fit3D 등 공개 데이터 활용

**법적 요구사항:**
- 개인정보보호법: 필수 동의(영상 촬영·리포트 전송) / 선택 동의(AI 학습 활용) 분리
- 비식별 처리: 포즈 키포인트 추출 후 원본 삭제 가능
- 회원 열람·삭제 권리 보장
